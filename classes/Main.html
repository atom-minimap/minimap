    Overview

    Public: The `Minimap` package provides an eagle-eye view of text buffers. It also provides API for plugin packages that want to interact with the minimap and be available to the user through the minimap settings. ## Plugins Interface Plugins should conform to the following interface: ```coffee class Plugin void activatePlugin: -> void deactivatePlugin: -> bool isActive: -> ```
